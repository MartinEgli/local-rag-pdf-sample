Helvetia Alpine Sensor Network Overview

Fictional document created for the Local RAG PDF sample.

Purpose

The network monitors air temperature and relative humidity at fictional alpine field stations.

Operating Facts

FactConfigured value
Upload intervalEvery 15 minutes
Offline gateway buffer72 hours of sensor readings
TransportMQTT over TLS 1.3
Gateway retention after upload30 days
Low battery alertBelow 20 percent

Outage Behaviour

During a network outage, each gateway buffers readings locally for up to 72 hours. When connectivity returns, the gateway uploads the oldest buffered reading first.
